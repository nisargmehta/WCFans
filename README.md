# WCFans

WCFans is a responsive FIFA World Cup 2026 fan hub built with React and Tailwind CSS. It includes a live scores ticker, expandable match cards, a World Cup news feed, and a playful haircut tracker that follows the five-wins-in-a-row fan challenge.

The app currently uses mock data, with the 2026 schedule and teams sourced from the `openfootball/worldcup.json` project. The client and mock server data layers are separated so a live REST feed can be introduced later without reshaping the UI.

## Local Development

```bash
npm install
npm run dev
```

## Checks
